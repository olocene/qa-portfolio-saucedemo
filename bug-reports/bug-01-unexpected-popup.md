# Bug Report – Unexpected Password Popup

## Description
Unexpected popup suggesting password change appears after successful login.

## Preconditions
- User on login page

## Steps to Reproduce
1. Go to https://www.saucedemo.com/
2. Enter valid credentials:
   - Username: standard_user
   - Password: secret_sauce
3. Click Login

## Expected Result
- User is redirected to inventory page without interruptions

## Actual Result
- Login successful
- Popup displayed suggesting password change

## Severity
Medium

## Priority
Medium
